Instructions pour créer vos images de certifications :

  • Téléchargez vos certificats depuis Coursera, LinkedIn Learning, etc.
  • Convertissez en JPG si nécessaire (format recommandé)
  • Redimensionnez à 400x300px pour une taille optimale
  • Nommez selon le format : [issuer]-[technology].jpg
  • Placez dans le dossier : public/certifications/images/

Exemples de noms de fichiers :

  • ibm-containers-kubernetes.jpg
  • whizlabs-azure-networks.jpg
  • upenn-java-oop.jpg
  • meta-react-basics.jpg
  • umich-javascript-interactivity.jpg
